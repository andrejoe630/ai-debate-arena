## Tips for Using the Interface

1. **Start Simple**: Just type a topic and press Enter in auto-mode
2. **Check Console**: Press F12 to see debug logs if something's wrong
3. **Mode Toggle**: Use the tabs at the bottom to switch modes
4. **Manual Control**: Click the Auto button if you want to pick specific AIs
5. **Watch Progress**: The status bar shows what's happening in real-time
